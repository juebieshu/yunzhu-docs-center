# 概述

本节收集了那些在游玩过程中可能会遇到的问题，以及解决方法。

**请在询问之前善用搜索功能，因为您遇到的问题可能已经被解决过**。以免给他人带来困扰。

::: info
本节更新缓慢，如果遇到无法解决的问题，请到门户群询问。
:::

导出错误报告，我们才能更快帮助您定位问题。
:::
